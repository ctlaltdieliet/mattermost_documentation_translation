Atlassian-tools
== == == == == == == == == == == == == == == == == == == == == ==

Onze ` Integrations Directory <https://integrations.mattermost.com> ` _ heeft verschillende integraties om Atlassian tools in Matterbest te verbinden.

Hier zijn enkele populaire opties hieronder, met inbegrip van self-hosted on-prem of zelf-gehoste private cloud-oplossingen, en vendor-gehoste SaaS-oplossingen.

Jira Plugin ~ ~ ~ ~ ~ ~ ~ ~ ~ n~ ~ ~ ~ ~ ~ ~ ~ n~ ~ ~ ~ ngeren

-Twee-weg integratie tussen Mattermore en Jira, ontwikkeld door Matterbest.
-Stuur eventberichten van uw Jira-projecten naar Mattermeeste kanalen, met volledige JQL filtering ondersteuning.
-Maak en transitie Jira problemen, en voeg Mattermeeste berichten aan Jira in de Mattermeeste user interface.
-Ondersteunt de producten van Jira Core en Jira Software, voor Cloud-, Server-en Data Center-platforms.
-Broncode + docs: https://github.com/mattermost/mattermost-plugin-jira
