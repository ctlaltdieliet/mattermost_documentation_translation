== == == == == == == == = Gegevens bulkladen
== == == == == == == == =

Grote hoeveelheden gegevens kunnen worden geïmporteerd uit een 'JSONL'
<https://jsonlines.org> ` __ bestand in Matterbest op de opdrachtregel met behulp van de functie voor bulkladen. Deze functie is het meest geschikt voor het migreren van gegevens van een bestaand messagingsysteem of voor het vooraf vullen van een nieuwe installatie met gegevens.

U kunt de volgende gegevenstypen importeren:

-Teams
-Kanalen (openbaar en privé)
-Gebruikers
-Teamlidmaatschappen van gebruikers
-Gebruikers ' Kanaal lidmaatschappen
-meldingsvoorkeuren voor gebruikers
-Ambten (vaste, niet-repliek)
-Posts ' antwoorden
-Reacties
-Bestand Bijlagen posten
-Directe berichten-en groepsberichtenkanalen
-Directe berichten en groepsberichten
-Directe berichten van een gebruiker naar zichzelf
-Machtigingen
-Aangepaste Emoji
